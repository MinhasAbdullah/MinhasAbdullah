## Projects

### 🏠 Roshan Aashiyana — Smart Property Marketplace
> *Full-stack real estate platform built for Pakistan · In active development*

A platform that replaces Pakistan's offline property search with a centralised, secure digital marketplace.

- Dual account system — normal users + verified dealers with email verification and role-based access
- AI-assisted property description generator (external AI API integration)
- Advanced search: city, area, type, price range, rooms, purpose
- Wishlist, reviews & ratings, admin dashboard with analytics
- Multi-image upload, inquiry management, automated SMTP email notifications
- Property status lifecycle: Available → Sold / Rented

`Django` `PostgreSQL` `Django Auth` `SMTP` `AI API` `Django ORM`
